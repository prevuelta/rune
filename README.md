# Rune
## Work in progress

SVG glyph creator using paper.js. Grid based design system to produce letter forms, icons etc then export as SVG from browser.

## To run:

Uses npm as a taskrunner

Basic task to build / watch:

```
npm run build:watch
```

Enjoy :)
